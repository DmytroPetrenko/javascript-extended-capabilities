Создайте приложение, в котором пользователь будет отправлять запросы по адресу: ‘https://jsonplaceholder.typicode.com/’ и получать пользователя по id, который будет передан в функцию. В случае, если пользователь не найден – отобразить ошибку. Если найден – то происходит отправка второго запроса с id пользователя по данному адресу для нахождения комментариев. Комментарии необходимо отобразить в тело документа. Используйте функцию setTimeout для генерации задержки и async/await при создании функций.
